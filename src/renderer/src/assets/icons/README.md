# Icons

This folder contains all the icons used in the project.

## Source

All icons were downloaded from [https://iconos8.es/](https://iconos8.es/), a resourceful site for high-quality icons.

## Style

The icon style used in this project is **Outline** (not **Filled**).
Filled-style icons are avoided to maintain visual consistency.

## Adding New Icons

If you want to add new icons to this folder:

1. Go to [https://iconos8.es/](https://iconos8.es/)
2. Search for the desired icon.
3. Make sure to select the **Outline** style before downloading.
4. Download the icon (preferably in SVG or PNG format).
5. Add it to this folder.

Please keep the icon style consistent across the project.
